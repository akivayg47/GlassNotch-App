# GlassNotch

GlassNotch is a personal fork of Boring Notch with a configurable Liquid Glass appearance system.

## Added in this build

- GlassNotch branding and app display name
- Ten glass presets: Clear, Frosted, Smoked, Crystal, Blue, Purple, Green, Adaptive, High Contrast, and Custom
- Live controls for opacity, blur, glow, saturation, and custom tint
- Existing Launch at Login option retained in General settings
- All original Boring Notch features and layout remain intact

## Build

Open `boringNotch.xcodeproj` in Xcode, select the main app target, choose your Mac, and press Run.
